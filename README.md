#Flat-File CMS

##What we want ? 
- Open Source
- No Database, based on files
- Markdown
- Archivable
- Media friendly
- Admin 
- Based on plugin 
- Basic Template 
- Cache system
- Great Documentation
- Good template system 
- Tags system
- Multlinguage
- Search plugin
